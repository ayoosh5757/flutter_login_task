المشروع الأول: Luna Stack
​الوصف: تطبيق بسيط يشرح أساسيات التنقل (Navigation) في Flutter.
​📝 المتطلبات المنفذة:
​Home & Details: تصميم شاشتين بتنسيق وردي ولافندر.
​Push & Pop: التنقل للأمام وللخلف باستخدام Navigator.
​Interactivity: إضافة أزرار تفاعلية سحرية.
